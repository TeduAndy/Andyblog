Author: Andy Date: July 22, 2022  8:35:32 Category: Blog-Hexo框架

GitHub 產生站點專案

1. 先點 Github 官網跟著圖片點

第一步 點擊 repositories


第二步 點new 產生站點專案


第三步 在橘色文字那邊輸入 你的用戶名 + github.io 然後按 create repository


第四步 點擊 Your repositories 點選 剛創建的專案 點選 code 會得到 部屬會用到網址

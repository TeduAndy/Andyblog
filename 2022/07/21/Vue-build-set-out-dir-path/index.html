VUE build 打包輸出修改位置

Author: Andy Date: July 21, 2022  14:03:30 Category: Vue

說明:

  • 輸出時 打包位置 改去別的地方

使用方法:

  • 在 vite.config.js 配置文件裡面 的 export default defineConfig 添加這段 build:{outDir: ‘輸出的位置路徑’}, 打包時位置便會到輸入的路徑裡面
